# Manual do Usuário: Portal de Atualização Baldez (V6.1) 🏛️📖

Bem-vindo ao Portal de Refinamento de Conteúdo da **Baldez Advogados**. Este guia explica como utilizar o formulário para garantir que os dados cheguem corretamente ao site principal.

## 👥 Seção: Equipe de Especialistas
Para cada advogado, preencha os 5 campos obrigatórios:
1.  **Nome Completo:** Nome que aparecerá no card.
2.  **Cargo:** Ex: Sócio Fundador, Advogado Associado.
3.  **Número OAB:** Registro profissional.
4.  **Bio Curta:** Resumo de 2 linhas para o card inicial do site.
5.  **Perfil Completo:** Biografia detalhada que aparecerá quando o cliente clicar em "Ver Perfil".

**Ação:** Clique em "Inserir Advogado" para cada membro. Eles serão listados abaixo do formulário.

## 🎯 Seção: Áreas de Atuação
O escritório pode listar quantas especialidades desejar.
1.  **Área:** Título (ex: Direito Civil).
2.  **Descrição:** O que o escritório faz nessa área.

**Ação:** Clique em "Inserir Área" para adicionar à lista oficial.

## 📞 Validação e Envio
Ao final, insira seu E-mail ou WhatsApp para que o Squad da AIA possa validar os dados.
- Clique em **"Finalizar e Consolidar Dados"**.
- Aguarde a tela verde de sucesso.
