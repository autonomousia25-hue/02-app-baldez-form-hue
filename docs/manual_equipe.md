# Manual do Portal de Refinamento: Baldez Advogados 🏛️

Este documento resume as funcionalidades do novo portal de atualização de conteúdo e orienta a equipe sobre o fluxo correto de envio.

## 🔗 Acesso ao Portal
O formulário está disponível publicamente em:
**[https://02-app-baldez-form-hue.vercel.app/](https://02-app-baldez-form-hue.vercel.app/)**

---

## 🛠️ Principais Funcionalidades

### 1. Gestão de Equipe (Múltiplos Especialistas)
Agora é possível enviar toda a equipe em uma única submissão:
1. Preencha os campos (Nome, OAB, Bio) na seção **Equipe de Especialistas**.
2. Clique no botão **"Inserir Especialista na Lista"**.
3. O membro aparecerá em uma lista visual. Repita o processo para todos os advogados.
4. Somente após adicionar todos, clique no botão final de envio na parte inferior da página.

### 2. Máscara de WhatsApp
O campo de telefone formata automaticamente para o padrão brasileiro `(00) 00000-0000`.

### 3. Heurísticas de Nielsen (Dicas Visuais)
Abaixo de cada campo, incluímos pequenas dicas (H2, H4, H10) que ajudam a manter a qualidade editorial do site.
